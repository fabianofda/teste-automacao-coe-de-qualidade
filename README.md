## Sobre

Repositório teste-automacao-coe-de-qualidade e2e

## Stacks
- Cypress
- Javascript

## Rodando

1. Clonar o repositório, instalar as dependências
```
npm install
```

2. Subir o Cypress UI
```
npx cypress open 
```

3. Executar testes em Headless
```
npx cypress run 
```
